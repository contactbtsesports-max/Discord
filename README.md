# Discord Auto-Moderation Bot

**Official Discord Auto-Moderation Bot** - Automatically detects and blocks insults, profanity, and rule violations in real-time.

## Features

✅ **Automatic Moderation** - No commands needed
✅ **Real-time Detection** - Instantly blocks violations
✅ **Insult & Profanity Filtering** - Detects and removes offensive content
✅ **Rule Enforcement** - Blocks content that violates server rules
✅ **Multi-Language Support** - Python and JavaScript versions
✅ **Easy Setup** - Simple configuration and deployment

## Available Versions

### Python Version
Location: `python/`
- Uses: `discord.py`
- Python 3.8+

### JavaScript Version
Location: `javascript/`
- Uses: `discord.js`
- Node.js 16+

## Quick Start

### Python
```bash
cd python
pip install -r requirements.txt
python main.py
```

### JavaScript
```bash
cd javascript
npm install
node main.js
```

## Configuration

1. Create a `.env` file in the respective folder
2. Add your Discord Bot Token:
   ```
   DISCORD_TOKEN=your_bot_token_here
   ```
